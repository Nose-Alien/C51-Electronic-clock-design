# C51电子时钟系统

基于STC89C52单片机的电子时钟系统，具有LCD1602显示、按键交互、闹钟提醒、整点语音报时、12/24小时制切换等功能。

## 功能特性

- 实时显示年、月、日、时、分、秒及星期
- 支持12/24小时制切换
- 闹钟设置（时、分、开关）
- 整点报时（8:00~22:00，支持12/24小时制语音播报）
- 通过串口控制语音模块（虚拟串口可仿真）
- 按键操作：确认、菜单、加、减、左移、右移、时制切换
- Proteus仿真支持

## 硬件平台

- 主控芯片：STC89C52RC
- 显示模块：LCD1602（8位并口）
- 输入模块：7路独立按键
- 报警模块：有源蜂鸣器
- 语音模块：串口控制（如SYN6288，仿真可用虚拟串口模拟）

## 软件结构

- `src/`：主程序源代码
- `include/`：头文件（LCD1602、MatrixKey、SpeechDriven、Delay等）
- `lib/`：硬件驱动库
- `proteus/`：Proteus仿真工程文件
- `.pio/`：PlatformIO编译输出（含HEX文件）
- `VSPD虚拟串口 6.9 汉化版/`：虚拟串口工具，用于仿真语音模块
